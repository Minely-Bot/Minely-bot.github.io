100% Free Forever Intermittent Hosting (No 24/7) Prefix: mc!

About the Bot

Engineered using rapid, optimized JavaScript architecture to keep performance light and query data transmission quick.

Generates real-time custom status cards and authentic Minecraft-styled tooltip boxes natively inside your Discord channels.

Server Diagnostics

Instantly fetch operational statistics, software targets, plugin configurations, and online player counts from public server IPs.

Frequently Asked Questions

Why is Minley offline right now?

Minley runs on a custom, complimentary deployment tier that does not promise fixed 24/7 background run-times. If commands are unresponsive, it means the host cluster is currently resting or cycling operations. Check back soon!

Is there a Premium configuration level available?

No. Minley is open access and distributed 100% free of charge. Every configuration command, graphics render, and checking feature is entirely open for public application without paywalls.

How can I target a specific server using diagnostic tracking?

For optional [ip] parameters, simply type your server's domain identifier directly following the targeted command expression (e.g., mc!status mc.hypixel.net).
